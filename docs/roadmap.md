# Roadmap

> **Stub:** This roadmap establishes sequence, not deadlines.

## Phase 1 — Foundation

- [x] Establish the Project RUN working name.
- [x] Document guiding principles and work areas.
- [x] Define an initial metadata model.
- [x] Create a repository structure.
- [ ] Select licenses for original text, code, and metadata.
- [ ] Add validation and formatting checks.

## Phase 2 — Representative cataloging

Catalog a small, varied sample before attempting a complete inventory:

- [ ] one periodical issue;
- [ ] one computer or console;
- [ ] one peripheral;
- [ ] one boxed software title;
- [ ] one loose disk or tape;
- [ ] one manual or book;
- [ ] one object with a meaningful personal story.

Use these records to revise the schema based on actual needs.

## Phase 3 — Collection workflow

- [ ] Define identifiers and labeling conventions.
- [ ] Define photography standards.
- [ ] Define condition vocabulary.
- [ ] Define storage-location notation.
- [ ] Document acquisition and disposition procedures.
- [ ] Develop a lightweight inventory interface or import tool.

## Phase 4 — Archive connections

- [ ] Link periodicals to Internet Archive and library records.
- [ ] Establish table-of-contents indexing conventions.
- [ ] Record software and manual surrogates.
- [ ] Add source-quality and rights-status fields.
- [ ] Identify materials with no adequate digital surrogate.

## Phase 5 — Workbench procedures

- [ ] Document safe hardware intake and testing.
- [ ] Document removable-media imaging workflows.
- [ ] Track repairs, modifications, and test results.
- [ ] Establish capture standards for screenshots, audio, and video.
- [ ] Create reproducible experiment and restoration logs.

## Phase 6 — Public access

- [ ] Decide what information must remain private, such as precise storage location or personal contact data.
- [ ] Generate a searchable public catalog.
- [ ] Create accessible item pages and narrative exhibits.
- [ ] Offer structured metadata downloads.
- [ ] Publish research and creative outputs.

## Questions to resolve

- What is the appropriate long-term scope: personal collection, collaborative archive, or both?
- Which controlled vocabularies are worth adopting?
- How should market value be recorded without making it the organizing principle?
- What materials can be redistributed, and under which licenses?
- What level of technical conservation is realistic?
- Which stories should become the first public exhibits?
