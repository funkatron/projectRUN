# Naming

## Working name: Project RUN

The project needs a name before it needs a permanent brand. **Project RUN** is the current working name and repository identity.

`RUN` fits the project in two ways:

1. In home-computer culture, `RUN` turns entered or loaded instructions into action.
2. More broadly, the collection is not the endpoint. Research, restoration, interpretation, and creation are forms of execution.

The name is intentionally concise and provisional enough to support experimentation. It should not be expanded into an artificial acronym unless a genuinely useful meaning emerges.

## Usage

- Human-readable name: **Project RUN**
- Repository name: `projectRUN`
- Internal slug, where needed: `project-run`
