# Periodicals

Periodical records are organized first by publication slug, then by cover date or another stable issue identifier.

```text
collections/periodicals/
└── run/
    ├── 1988-01.yaml
    └── 1988-01/
        ├── front-cover.jpg
        ├── back-cover.jpg
        └── spine.jpg
```

The media directory shown above is a convention; images have not yet been committed.

Issue-level records may later connect to separate article, advertisement, type-in program, review, author, company, and product records. Do not embed a complete page-level index until the representation has been tested on several publications.

## Known collection scope

The physical collection includes issues from approximately 1987–1992, with titles including:

- Commodore Magazine
- RUN
- INFO
- Game Players
- GamePro
- VideoGames & Computer Entertainment
- Electronic Gaming Monthly

Condition varies. Digital versions often exist through Internet Archive and should be linked where they can improve discovery and search.
